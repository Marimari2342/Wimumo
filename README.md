# Wimumo

Proyecto orientado a la exploración de bioseñales y su transformación en sistemas interactivos.

## Descripción

Wimumo es un sistema que permite capturar señales del cuerpo y transformarlas en datos para su procesamiento y visualización, explorando su uso en experiencias interactivas.

## Tecnologías

- Java
- Processing / p5.js
- Git

## Mi aporte

- Refactorización del código para mejorar la modularidad y reducir duplicación.
- Aplicación de POO (encapsulamiento, polimorfismo).
- Reorganización de la estructura del sistema.
- Propuesta de estrategias para la recolección y análisis de datos.
- Exploración de patrones en bioseñales para futuros modelos de IA.

## Estado del proyecto

En desarrollo. Actualmente enfocado en la mejora de la estructura del código y la exploración de datos.

## Próximos pasos

- Implementación de procesamiento de datos más avanzado.
- Análisis de patrones en señales.
- Integración con modelos de IA.
